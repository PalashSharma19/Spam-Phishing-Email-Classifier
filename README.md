# Hybrid Spam & Phishing Detection System

This project implements a hybrid approach combining rule-based detection
with machine learning models to identify spam and phishing messages.

## Features
- Rule-based expert scoring system
- ML-based text classification
- Hybrid decision logic
- Explainable predictions

## Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF)
- SQLite / MySQL

## Project Status
Rule engine implementation in progress.
